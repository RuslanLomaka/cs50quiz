# Capstone Project (CS50 Web)

This repository contains my final project for **CS50’s Web Programming with Python and JavaScript**.

## Project Idea

The goal of this project is to build a flexible web platform that allows users to create **very detailed and fully customized quizzes** in just a few prompt edits and a couple of mouse clicks.

Instead of manually designing questions one by one, users can rely on **AI or other external tools** to generate structured quiz content, quickly refine it, and immediately publish it for real users to complete.

The core idea of the project is:
- speed of quiz creation  
- high level of customization  
- creative freedom without technical complexity  

## Main Features (Planned / In Progress)

- User registration and authentication
- Quiz creation using structured JSON input
- AI-assisted quiz generation workflow
- Quiz completion and automatic scoring
- Public visibility of results after quiz completion
- Named users and guest participants
- Comments and feedback for quizzes and individual questions
- Admin dashboard
- Unique visitors tracking (per day)
- Basic charts and usage statistics
